# Dự báo doanh thu phòng vé phim điện ảnh

## Mô tả
Dự án xây dựng mô hình dự báo doanh thu phim dựa trên ngân sách và dàn diễn viên.

## Dataset
TMDB 5000 Movies Dataset  
File: tmdb_5000_movies.csv

## Bài toán
- Loại bài toán: Hồi quy (Regression)  
- Mục tiêu: Dự báo doanh thu (revenue)

## Feature sử dụng
- budget (ngân sách)
- cast_count (số lượng diễn viên)

## Phương pháp
- Linear Regression

## Đánh giá
- Mean Absolute Error (MAE)

## Cách chạy
python src/model.py
